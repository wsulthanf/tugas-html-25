Mengenal Semantic Element di HTML5

  • Article

    • Ketika ingin membuat layout untuk menampilkan block-block artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer karena tag article memiliki type block, artinya element ini memiliki lebar sesuai browser jika kita tidak menentukan lebarnya.

  • Aside

    • Element ini bisa kita gunakan untuk melengkapi layout utama. Misal kita memiliki halaman blog maka kita bisa menambahkan aside di bagian bawah postingan. Karena aside memiliki sifat type/sifat block maka lebarnya sesuai page jika kita tidak menentukan lebarnya.
